# 14-Key Left-Hand Optimized Layout (Halmak-Style)

This layout is designed for high-speed one-handed typing using only the left-hand grid (**Q-T, A-G, Z-V**). 

## Keyboard Logic
- **Base Layer:** Tap the key for the most frequent letters (94% coverage).
- **Space Layer:** Hold **Spacebar** with your thumb while tapping the key.

## 3. High-Frequency Sequences
Use these motions to master the most common English patterns:

### Vowel Sequences
- **IO:** Middle (I) → Ring (O) on the **Bottom Row**.
- **EU:** Middle (E) on **Home Row** → Index (U) on **Top Row**.
- **EA:** Pinky (E) → Ring (A) on the **Home Row**.

### Common Words
- **THE:** Index+ (T) [Home] → Index (H) [Bottom] → Middle (E) [Home].
- **ION:** Middle (I) [Bottom] → Ring (O) [Bottom] → Index+ (N) [Bottom].
- **AND:** Pinky (A) [Home] → Index+ (N) [Bottom] → Ring (D) [Top].
