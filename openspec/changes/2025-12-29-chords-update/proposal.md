# Proposal: Update Chords for Backspace, UI Actions, and Symbols

## Goal
Update Kanata chords to provide a comprehensive set of actions for editing, navigation, symbols, and formatting on the Halmak and Mirror layers.

## Changes

### 1. Halmak Layer Chords (`halmak-chords`)
- `ao` (Physical `s` + `f`): Backspace.
- `aio` (Physical `s` + `d` + `f`): Ctrl + Backspace.
- `ei` (Physical `a` + `d`): Delete.
- `dn` (Physical `x` + `c`): Ctrl + J (New Line).
- `ldnm` (Physical `z` + `x` + `c` + `v`): Escape.
- `eo` (Physical `a` + `f`): Copy (Ctrl + C).
- `ch` (Physical `q` + `r`): `{` (Shift + [).
- `au` (Physical `s` + `g`): `(` (Shift + 9).
- `sr` (Physical `w` + `t`): `[` (Standard [).
- `sth` (Physical `w` + `e` + `r`): `.`
- `hs` (Physical `r` + `w`): `,`
- `ot` (Physical `f` + `e`): `;`
- `ct` (Physical `q` + `e`): `<` (Shift + ,).
- `tr` (Physical `e` + `t`): `>` (Shift + .).
- `es` (Physical `a` + `w`): `"` (Shift + ').
- `ato` (Physical `s` + `e` + `f`): `=`
- `ou` (Physical `f` + `g`): `_` (Shift + -).

### 2. Mirror Layer Chords (Hold Space) (`mirror-chords`)
- `aio` (Physical `s` + `d` + `f`): Ctrl + Alt + Backspace.
- `eo` (Physical `a` + `f`): Paste (Ctrl + V).
- `ch` (Physical `q` + `r`): `}` (Shift + ]).
- `au` (Physical `s` + `g`): `)` (Shift + 0).
- `sr` (Physical `w` + `t`): `]` (Standard ]).
- `ot` (Physical `f` + `e`): `:` (Shift + ;).
- `es` (Physical `a` + `w`): `'` (Single Quote).
- `ato` (Physical `s` + `e` + `f`): `+` (Shift + =).
- `ou` (Physical `f` + `g`): `-` (Standard -).

## Reason
To provide a highly efficient one-handed typing experience by mapping frequently used symbols and actions to intuitive finger chords.
